In Part A, I crafted a responsive webpage from scratch using pure HTML and CSS. The creative process was like painting a canvas with code, allowing me to have complete control over the design. Creating custom media queries and styles for various screen sizes presented a delightful challenge, particularly when perfecting the card deck layout. Although it demanded more time and effort, the result was a visually engaging and unique webpage.

On the other hand, in Part B, I recreated the same webpage using the Bootstrap framework. Bootstrap's pre-defined components and grid system sped up development, reducing the need for extensive custom CSS. However, understanding and applying Bootstrap classes effectively required mastering the Bootstrap magic. This method undoubtedly suited projects with standard components and layouts, making it a convenient choice when time is of the essence. The choice between the two approaches depends on project requirements and the need for customization, both having their unique appeal and benefits.
